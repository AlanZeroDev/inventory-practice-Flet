#Inventory Manager (Flet)

Aplicación simple de gestión de stock desarrollada en Python usando Flet, Pandas y SQLite.

## Funcionalidades
- Carga de productos
- Control de cantidades
- Selección de ubicación
- Interfaz gráfica simple

## Tecnologías
- Python
- Pandas
- Flet
- SQLite
